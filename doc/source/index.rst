AutoNetkit
======================================

AutoNetkit allows you to create and deploy large-scale emulated networks based on the `Netkit <http://www.netkit.org>`_ emulation platform.

This documentation is also available in `PDF format <AutoNetkit.pdf>`_ 

                                                                            

Features
________ 

.. image:: images/multi_as.*

Fast
	It takes **less than 20 seconds** to configure, plot and deploy the network above  
	
Flexible    
	Services including Quagga and BIND can be configured automatically, enabling fast, realistic and affordable network experimentation
                                                                                                                                      
Easy to use
	Either through configuration files or using the Python API
	
Automated
	Configuration, plotting and deployment are handled automatically    
	
Available   
	AutoNetkit is written in Python, and available from the `Python Package Index <http://pypi.python.org/pypi/AutoNetkit/>`_.
